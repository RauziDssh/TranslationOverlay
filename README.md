# TranslationOverlay

TranslationOverlay は、画面上の特定の領域をキャプチャし、OCR（文字認識）と翻訳APIを組み合わせて、翻訳結果を画面上に直接オーバーレイ表示するツールです。

ゲーム、動画、またはコピーできないアプリケーションのテキストを素早く理解するために設計されています。

## ✨ 主な機能

- **リアルタイム翻訳**: 画面範囲内のテキストを自動的に検出して翻訳します。
- **オーバーレイ表示**: 元のテキストの上に、または指定した位置に翻訳後のテキストを重ねて表示します。
